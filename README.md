艾瑞斯驱动安装：
.net 控制台项目，快速索引驱动，按需安装。

参数帮助(参数为小写):

	/auto            自动索引驱动，自动安装驱动。
                     注：如不指定此参数则为自动索引，手动确认安装。

    /offlineosdir=   对离线的系统安装安装驱动，例：/offlineosdir=D:\
                     注：对当前系统安装驱动无需使用此参数。

    /zippwd=         为驱动包指定解压密码，明文/zippwd=密码，密文/zippwd=*密码
                     注：此参数仅支持以参数形式指定。

    /*zippwd=        加密明文，输出一个加密后的密码

    /setdrvdir=      设置驱动目录，程序有默认目录一般是不需要这个的。

    /logfile=off      使用此参数不输出日志文件。默认是输出的。


生成驱动压缩包的索引文件：

     生成此文件能极大提高驱动索引速度，强烈推荐。
     不生成也不影响安装，但索引驱动包会很慢。
     注：不以/auto参数运行就显示此选项，需每次添加或更新驱动执行一次。

输出离线驱动包支持硬件型号：

     注：不以/auto参数运行就显示此选项(需先生成驱动包索引文件)。

关于程序驱动压缩包提取说明：

     程序所在目录下有\Res目录，此目录可删除，但不建议。
     例：\Res\7z.exe 软件默认用7z解压缩提取，因为他快。
     但本程序并不依赖于此，程序有自己的内置代码，
     如对压缩包有加密需求，请删除7z，避免密码传递暴漏风险。
     例：\Res\devcon.exe 程序并不首选此但作为备用，可删除，但不建议。

关于程序驱动安装规则：

     对当前系统仅安装未安装驱动的设备或较新的驱动或已安装但存在问题的驱动
     对离线系统无此规则。
     全局规则，如多个驱动匹配同一设备，则默认选择版本较大的。

软件退出代码:

	0            正常退出/有驱动安装成功(具体日志会在控制台及日志中输出);
	-1           软件重复运行，多开错误;
	-2           软件驱动目录不存在;
	-3           参数指定的离线系统不是完整的系统(乱指定);
	-4           系统位宽不是x64;
	1607         系统版本过低(限制对<=1607版本安装驱动);
	-99          未能枚举到本设备硬件信息;
	-66          在驱动目录未找到任何驱动包或驱动文件;
	-88          在驱动包内未索引到本机设备驱动;
	-9999        索引到了驱动，但0个安装成功;

驱动安装输出类型：

    驱动安装成功；
    驱动安装成功，但重启生效；
    驱动安装失败，会返回一个错误代码；

日志输出：

    默认在程序所在目录输出日志文件DrvInst.log;
    当在挂载iso内直接使用时，因只读环境无法生成默认路径，
    会在任意磁盘根目录生成此日志文件。

关于驱动包：

    有DIY需求用户可自定义添加删除驱动包，
    但请注意每次更新后需执行一次"生成驱动索引文件"
